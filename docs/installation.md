# Installation Guide

Main source: http://www.github.com/synvek/synvek/releases

Mirror in China:  http://www.gitcode.com/synvek/synvek/releases

Select the latest version and install the corresponding installation program based on the operating system you are using

On Windows & Linux, you need to install Nvidia CUDA toolkit if you use CUDA backend support.

Vulkan backend can be used for most common disply cards.
